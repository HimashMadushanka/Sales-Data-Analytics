📊 Excel Sales Dashboard Project
📌 Project Overview

This project focuses on data cleaning, analysis, and dashboard creation using Microsoft Excel.
The goal is to transform raw data into meaningful insights through an interactive and visually appealing dashboard.

📂 Dataset
Source: Superstore Dataset (or your dataset name)
Records: ~10,000 rows
Features include:
Order Details (Order ID, Order Date, Ship Date)
Customer Information
Product Details
Sales, Profit, Discount, Quantity
🧹 Data Cleaning Steps

Performed using Excel functions & tools:

Removed duplicates
Handled missing values
Formatted date columns
Standardized product names
Created calculated columns (Profit Margin, etc.)
Converted raw data into structured table (Data_Table)
